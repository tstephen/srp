1. Architecture

The system is built 'API-first'. In other words the data is hosted on a cloud server and a series of services exposed to permit access to it. These services represent the Application Programming Interface (API) and support not only the web application at https://srp.digital but potentially other applications that may be built in the future by the community.

The front-end is built as a 'Progressive Web Application' which means that users need only a reasonably recent browser to access the system. The progressive part is that certain features, such as offline access, may only be available to the most modern browsers but a graceful fallback should exist back to at least Internet Explorer 10.

On the server side, the application is built on Spring Boot with a MySQL data store.
